  ## What I shipped

  Co-led delivery of a consumer health and wellness mobile application — full cross-functional squad: iOS, Android,
  Frontend, Backend, and QC. End-to-end ownership of squad health, technical quality, and incident response.

  A few things that happened along the way:

  - Contained 5 cascading production incidents triggered by a single deploy — postmortem led to structural process changes
  across staging, background jobs, and release gates
  - Coordinated MobSF security audits across both iOS and Android platforms; managed finding triage and grooming against
  active sprint work
  - Produced a full AWS infrastructure audit from scratch: service catalog, cost breakdown, risk surface, monitoring gaps,
  architecture diagram
  - Built the tooling layer on top: standup automation (Jira + Claude Code cron), gitflow audit CLI, and an AI-queryable
  team context wiki
